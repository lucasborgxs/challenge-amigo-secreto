# 🎁 Amigo Secreto

Este é um projetinho que fiz para treinar **HTML, CSS e JavaScript**.  
A ideia é bem simples: você adiciona os nomes dos amigos e o programa sorteia um deles como "amigo secreto".  

Foi uma forma de praticar lógica de programação e aprender a mexer melhor com o DOM no JavaScript.  

---

## 🚀 O que dá pra fazer

- Adicionar nomes em uma lista.  
- Impedir que o usuário adicione nomes vazios.  
- Mostrar todos os nomes que foram digitados.  
- Sortear aleatoriamente um amigo da lista.  

---

## 🛠️ Tecnologias

- **HTML** para a estrutura.  
- **CSS** para a parte visual.  
- **JavaScript** para a lógica do sorteio e manipulação da página.
